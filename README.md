# bun-template

Prerequisites:

- [`bun`](https://bun.com/)
- [`nvm`](https://github.com/nvm-sh/nvm)

To install dependencies:

```bash
nvm install
nvm use
bun install
```

To run:

```bash
bun run start
```

This project was created using `bun init` in bun v1.2.18. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
